# Obsidian Workspace ✦
> *"A focused mind deserves a beautiful space to think."*

A browser-based all-in-one productivity workspace — featuring notes, sticky notes, a countdown timer, stopwatch, and a live analog clock. No installs. No servers. No setup.

## 🌐 Live Demo

**[Try it here → obsidian-workspace.netlify.app](https://obsidian-workspace.netlify.app/)**

## ✨ Features

- 📝 Rich text notes with a masonry card grid layout
- 🗒️ Colorful sticky notes — Yellow, Pink, Mint, and Lavender themes
- ⏱️ Countdown timer with circular progress ring and audio alert
- ⏲️ Stopwatch with millisecond precision and full lap history
- 🕐 Smooth-sweeping SVG analog clock — always visible in the sidebar
- 💾 Full localStorage persistence — nothing is lost on refresh or close
- 🗑️ Inline delete confirmation — no accidental deletions
- 💻 Zero dependencies — single HTML file, no frameworks, no backend
- 🌐 Works offline after first load

## 🛠️ How It Works

1. Open the app — your workspace loads instantly with all your saved data
2. Go to **Notes** → write a title and body → click **Save Note**
3. Go to **Sticky Notes** → pick a color → click to edit inline
4. Go to **Timer & Stopwatch** → set a duration → hit Start — get an audio alert when done
5. The **analog clock** is always live in the sidebar, no matter which section you're in

## 🔧 Tech Stack

- React 18 — loaded via CDN, no npm needed
- Babel Standalone — JSX compiled directly in the browser
- Web Audio API — timer beep with no audio files
- Programmatic SVG — analog clock drawn entirely in code
- Browser localStorage — all data saved automatically
- Google Fonts — Playfair Display · DM Mono · Nunito
- Pure CSS animations — no animation libraries

## 🎨 Design

Built around a deep navy and electric cyan dark-mode aesthetic — premium, focused, and distraction-free. Frosted glass cards, subtle grain texture, cyan glow shadows, and smooth transitions throughout.
